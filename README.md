A narrative-driven, turn-based PWA RPG set in the decaying remains of Aurelune.

> "In a land where dawn was murdered, even a candle is an act of rebellion."

This project leverages **Gemini 3 Flash** to drive a fluid, "Dungeon Master" style storytelling experience. Built as a lightweight Progressive Web App (PWA), it combines deterministic d20-based mechanics with deep, grounded prose, focusing on the domestic realism of survival in a kingdom of eternal night.

## 🛠️ The Architecture
- **Frontend:** Next.js 14+ (App Router) / TypeScript
- **Styling:** Tailwind CSS (Theme: Charcoal & Ember)
- **AI Engine:** Gemini 3 Flash (via Google AI SDK)
- **State:** Zustand (Local Game State)
- **Manifest:** PWA compliant for mobile/desktop installation

## 📜 The Hook
The Ever Sun has failed. Archbishop Lazaruz has turned the Church of the First Dawn into a gateway for the restless dead, and King Aldric reigns as an infested shadow. You are a 'Living Spark' in a world of ash.
