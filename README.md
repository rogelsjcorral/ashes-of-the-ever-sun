A narrative-driven, turn-based PWA RPG set in the ruined kingdom of Aurelune.

> "In a land where dawn was murdered, even a candle is an act of rebellion."

Ashes of the Ever Sun is a dark fantasy role-playing game built as a lightweight Progressive Web App. It blends deterministic d20-based mechanics with AI-assisted narrative flow to create a Dungeon Master style experience rooted in survival, consequence, and the slow decay of a kingdom swallowed by eternal night.

## 📜 The Hook

The Ever Sun has failed. Archbishop Lazaruz has corrupted the Church of the First Dawn into a gateway for the restless dead, and King Aldric now reigns as an infested shadow upon a broken throne. You are one of the last Living Sparks in a world of ash, grave bells, and dying faith.

## 📖 The Lore

Read the full history of the fall of Aurelune in [BACKSTORY.md](./BACKSTORY.md).

## ✨ Core Features

- Turn-based progression using deterministic d20-inspired mechanics
- AI-assisted narrative events and branching encounters
- Lightweight PWA design for desktop and mobile play
- Dark fantasy worldbuilding centered on ruin, survival, and moral choice
- Grounded prose with a grim, old world tone

## 🛠️ Architecture

- **Frontend:** Next.js 14+ with App Router and TypeScript
- **Styling:** Tailwind CSS
- **AI Engine:** Gemini Flash via Google AI SDK
- **State Management:** Zustand
- **Platform:** Progressive Web App for desktop and mobile installation
